# 决策者接口文档

## 接口概述

决策者接口为系统决策者提供数据统计和分析功能，包括仪表盘概览、上报记录查询、统计分析等。

## 基础信息

| 属性 | 值 |
|------|------|
| **基础路径** | `/api/decision-maker` |
| **认证方式** | JWT Token |
| **请求头** | `Authorization: Bearer <token>` |

---

## 接口列表

| API路径 | 方法 | 功能描述 |
|---------|------|----------|
| `/dashboard` | GET | 获取仪表盘概览 |
| `/reports` | GET | 分页查询上报记录 |
| `/statistics` | GET | 获取月度统计 |
| `/statistics/trend` | GET | 获取统计趋势 |
| `/statistics/region` | GET | 获取区域统计 |

---

## 接口详情

### 1. 获取仪表盘概览

**路径**: `GET /api/decision-maker/dashboard`

**功能**: 获取系统整体数据概览，包括上报统计、AQI统计、区域排名等。

**请求参数**: 无

**响应示例**:

```json
{

    "message": "success",
    "data": {
        "totalReports": 156,
        "pendingReports": 23,
        "confirmedReports": 128,
        "processedReports": 133,
        "avgAqi": 75.5,
        "maxAqi": 234,
        "minAqi": 23,
        "excellentDays": 12,
        "goodDays": 15,
        "pollutionDays": 3,
        "topRegions": [
            {
                "province": "山东省",
                "city": "济南市",
                "reportCount": 45,
                "avgAqi": 82.3
            },
            {
                "province": "山东省",
                "city": "青岛市",
                "reportCount": 38,
                "avgAqi": 68.7
            }
        ],
        "monthlyTrend": [
            {
                "month": "2026-04",
                "reportCount": 42,
                "avgAqi": 72.1
            },
            {
                "month": "2026-05",
                "reportCount": 56,
                "avgAqi": 78.3
            },
            {
                "month": "2026-06",
                "reportCount": 58,
                "avgAqi": 75.5
            }
        ]
    }
}
```

**响应字段说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| `totalReports` | Integer | 总上报数 |
| `pendingReports` | Integer | 待处理上报数 |
| `confirmedReports` | Integer | 已确认上报数 |
| `processedReports` | Integer | 已处理上报数 |
| `avgAqi` | BigDecimal | 平均AQI值 |
| `maxAqi` | Integer | 最高AQI值 |
| `minAqi` | Integer | 最低AQI值 |
| `excellentDays` | Integer | 优天数 |
| `goodDays` | Integer | 良天数 |
| `pollutionDays` | Integer | 污染天数 |
| `topRegions` | Array | 区域排名TOP5 |
| `monthlyTrend` | Array | 近3个月趋势 |

### 2. 分页查询上报记录

**路径**: `GET /api/decision-maker/reports`

**功能**: 分页查询上报记录，支持按省份、城市、状态筛选。

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `province` | String | ❌ | 省份名称 |
| `city` | String | ❌ | 城市名称 |
| `status` | String | ❌ | 状态：pending/assigned/detected/confirmed/unconfirmed |
| `pageNum` | Integer | ❌ | 页码，默认1 |
| `pageSize` | Integer | ❌ | 每页条数，默认10 |

**状态值说明**:

| 状态值 | 说明 |
|--------|------|
| `pending` | 待指派 |
| `assigned` | 已指派 |
| `detected` | 已检测 |
| `confirmed` | 已确认 |
| `unconfirmed` | 不合理 |

**响应示例**:

```json
{

                "address": "济南市历下区XX街道",
                "status": "confirmed",
                "statusDesc": "已确认",
                "pollutionType": "PM2.5",
```

### 3. 获取月度统计

**路径**: `GET /api/decision-maker/statistics`

**功能**: 获取指定月份的统计数据，支持按省份、城市筛选。

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `province` | String | ❌ | 省份名称 |
| `city` | String | ❌ | 城市名称 |
| `month` | String | ✅ | 月份，格式：yyyy-MM |

**响应示例**:

```json
{
    "code": 200,
    "message": "success",
    "data": [
        {
            "statId": 1,
            "province": "山东省",
            "city": "济南市",
            "statMonth": "2026-06",
            "totalReports": 45,
            "confirmedCount": 38,
            "avgAqi": 78.5,
            "maxAqi": 189,
            "minAqi": 32,
            "excellentDays": 8,
            "goodDays": 12,
            "lightPollutionDays": 6,
            "moderatePollutionDays": 3,
            "heavyPollutionDays": 1,
            "updatedAt": "2026-06-15T14:00:00"
        }
    ]
}
```

**响应字段说明**:

| 字段 | 类型 | 说明 |
|------|------|------|
| `statId` | Integer | 统计ID |
| `province` | String | 省份 |
| `city` | String | 城市 |
| `statMonth` | String | 统计月份 |
| `totalReports` | Integer | 总上报数 |
| `confirmedCount` | Integer | 已确认数 |
| `avgAqi` | BigDecimal | 平均AQI |
| `maxAqi` | Integer | 最高AQI |
| `minAqi` | Integer | 最低AQI |
| `excellentDays` | Integer | 优天数 |
| `goodDays` | Integer | 良天数 |
| `lightPollutionDays` | Integer | 轻度污染天数 |
| `moderatePollutionDays` | Integer | 中度污染天数 |
| `heavyPollutionDays` | Integer | 重度污染天数 |

### 4. 获取统计趋势

**路径**: `GET /api/decision-maker/statistics/trend`

**功能**: 获取指定时间范围的统计趋势数据。

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `province` | String | ❌ | 省份名称 |
| `city` | String | ❌ | 城市名称 |
| `startMonth` | String | ✅ | 开始月份，格式：yyyy-MM |
| `endMonth` | String | ✅ | 结束月份，格式：yyyy-MM |

**响应示例**:

```json
{
    "code": 200,
    "message": "success",
    "data": [
        {
            "statMonth": "2026-04",
            "totalReports": 42,
            "confirmedCount": 38,
            "avgAqi": 72.1
        },
        {
            "statMonth": "2026-05",
            "totalReports": 56,
            "confirmedCount": 52,
            "avgAqi": 78.3
        },
        {
            "statMonth": "2026-06",
            "totalReports": 58,
            "confirmedCount": 45,
            "avgAqi": 75.5
        }
    ]
}
```

---

### 5. 获取区域统计

**路径**: `GET /api/decision-maker/statistics/region`

**功能**: 获取指定区域的详细统计信息。

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `province` | String | ✅ | 省份名称 |
| `city` | String | ❌ | 城市名称（不传则返回全省统计） |

**响应示例**:

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "province": "山东省",
        "city": "济南市",
        "reportCount": 45,
        "avgAqi": 78.5
    }
}
```

## 错误响应格式

```json
{
    "code": 500,
    "message": "错误描述信息",
    "data": null
}
```

## 接口权限

所有决策者接口均需要决策者角色（role_id = 4）的用户才能访问。
