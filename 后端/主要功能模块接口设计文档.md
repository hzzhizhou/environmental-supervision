## 1. 公众监督员端接口

### 1.1 创建AQI上报

**功能描述**：公众监督员提交空气质量反馈信息

| 属性 | 值 |
|------|-----|
| **接口地址** | `/api/supervisor/reports` |
| **请求方式** | POST |
| **所属Controller** | SupervisorReportController |

**请求参数**：

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userId | Integer | 是 | 用户ID |
| aqiValue | Integer | 是 | AQI数值（0-500） |
| aqiLevel | String | 是 | AQI等级：优、良、轻度污染、中度污染、重度污染 |
| province | String | 是 | 省份名称 |
| city | String | 是 | 城市名称 |
| address | String | 否 | 详细地址 |
| remark | String | 否 | 备注描述 |
| pm25 | BigDecimal | 否 | PM2.5浓度（μg/m³） |
| pm10 | BigDecimal | 否 | PM10浓度（μg/m³） |
| o3 | BigDecimal | 否 | 臭氧浓度（μg/m³） |
| so2 | BigDecimal | 否 | 二氧化硫浓度（μg/m³） |
| no2 | BigDecimal | 否 | 二氧化氮浓度（μg/m³） |
| co | BigDecimal | 否 | 一氧化碳浓度（mg/m³） |

**请求示例**：
```json
{
    "userId": 1,
    "aqiValue": 120,
    "aqiLevel": "轻度污染",
    "province": "辽宁省",
    "city": "沈阳市",
    "address": "和平区XX街道XX号",
    "remark": "附近有明显异味",
    "pm25": 75.5,
    "pm10": 120.0
}
```

**成功响应**（HTTP 200）：
```json
{
    "code": 200,
    "message": "上报成功",
    "data": {
        "reportId": 1,
        "reportCode": "AQI-20260605-001",
        "userId": 1,
        "aqiValue": 120,
        "aqiLevel": "轻度污染",
        "province": "辽宁省",
        "city": "沈阳市",
        "status": "pending",
        "statusDesc": "待指派",
        "reportedAt": "2026-06-05T10:30:00"
    }
}
```

### 1.2 查询上报列表

**功能描述**：公众监督员查询自己的上报记录列表（分页）

| 属性 | 值 |
|------|-----|
| **接口地址** | `/api/supervisor/reports` |
| **请求方式** | GET |
| **所属Controller** | SupervisorReportController |

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userId | Integer | 否 | 按用户ID筛选 |
| status | String | 否 | 状态筛选：pending/assigned/detected/confirmed |
| province | String | 否 | 按省份筛选 |
| city | String | 否 | 按城市筛选 |
| keyword | String | 否 | 模糊搜索（地址/备注） |
| pageNum | Integer | 否 | 页码，默认1 |
| pageSize | Integer | 否 | 每页条数，默认10 |

```
GET /api/supervisor/reports?userId=1&pageNum=1&pageSize=10
```

### 2.2 用户管理-创建用户

**功能描述**：管理员创建新用户

| 属性 | 值 |
|------|-----|
| **接口地址** | `/api/admin/users` |
| **请求方式** | POST |
| **所属Controller** | AdminController |

**请求参数**：

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名（唯一） |
| password | String | 是 | 密码（明文，后台加密存储） |
| realName | String | 否 | 真实姓名 |
| phone | String | 是 | 手机号（唯一） |
| roleId | Integer | 是 | 角色ID（1-管理员，2-网格员，3-公众监督员，4-决策者） |
| gridId | Integer | 否 | 网格ID（仅网格员需要） |
| status | Integer | 否 | 状态，默认1（启用） |

**请求示例**：
```json
{
    "username": "supervisor_new",
    "password": "123456",
    "realName": "李四",
    "phone": "13800138007",
    "roleId": 3,
    "status": 1
}
```

**成功响应**（HTTP 200）：
```json
{
    "code": 200,
    "message": "用户创建成功",
    "data": {
        "userId": 6,
        "username": "supervisor_new",
        "realName": "李四",

```

### 2.3 任务指派

**功能描述**：管理员将公众上报指派给网格员处理

| 属性 | 值 |
|------|-----|
| **接口地址** | `/api/admin/tasks/assign` |
| **请求方式** | POST |
| **所属Controller** | AdminController |

**请求参数**：

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| reportId | Integer | 是 | 上报记录ID |
| adminId | Integer | 是 | 执行指派的管理员ID |
| workerId | Integer | 是 | 被指派的网格员ID |
| deadline | String | 否 | 截止时间（ISO8601格式） |

**请求示例**：
```json
{
    "reportId": 1,
    "adminId": 4,
    "workerId": 3,
    "deadline": "2026-06-06T18:00:00"
}
```

```json
{
    "code": 200,
    "message": "指派成功",
    "data": {
        "taskId": 1,
        "reportId": 1,
        "adminId": 4,
        "workerId": 3,
        "workerName": "王五",
        "assignTime": "2026-06-05T11:30:00",
        "deadline": "2026-06-06T18:00:00",
        "status": "assigned",
        "statusDesc": "已指派"
    }
}
```

### 2.4 审核确认

**功能描述**：管理员审核确认网格员提交的实测数据

| 属性 | 值 |
|------|-----|
| **接口地址** | `/api/admin/audit/{detectId}` |
| **请求方式** | PUT |
| **所属Controller** | AdminController |

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| detectId | Integer（路径参数） | 是 | 实测数据ID |
| auditorId | Integer | 是 | 审核管理员ID |
| action | String | 是 | 审核动作：confirm（确认）、reject（驳回） |
| result | String | 否 | 审核备注 |

**请求示例**：
```json
{
    "auditorId": 4,
    "action": "confirm",
    "result": "数据有效，确认通过"
}
```

**成功响应**（HTTP 200）：
```json
{
    "code": 200,
    "message": "审核完成",
    "data": {
        "detectId": 1,
        "confirmStatus": 1,
        "confirmStatusDesc": "已确认",
        "confirmedAt": "2026-06-05T14:00:00",
        "confirmNote": "数据有效，确认通过"
    }
}
```

|------|-----|

| **请求方式** | POST |
| **所属Controller** | GridWorkerController |

**请求参数**：

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| taskId | Integer | 是 | 任务ID |
| workerId | Integer | 是 | 网格员ID |

| aqiLevel | String | 是 | 实测AQI等级 |

| pm10 | BigDecimal | 否 | PM10实测值 |
| o3 | BigDecimal | 否 | 臭氧实测值 |
| so2 | BigDecimal | 否 | 二氧化硫实测值 |
| no2 | BigDecimal | 否 | 二氧化氮实测值 |
| co | BigDecimal | 否 | 一氧化碳实测值 |


```json
{
    "code": 200,
    "message": "提交成功，等待审核",
    "data": {
        "detectId": 1,
        "taskId": 1,
        "workerId": 3,
        "aqiValue": 115,
        "aqiLevel": "轻度污染",
        "detectTime": "2026-06-05T15:00:00",
```

## 统一响应格式

### 成功响应
```json
{
    "code": 200,
    "message": "success",
    "data": { ... }
}
```

### 失败响应
```json
{
    "code": 400,
    "message": "错误描述",
    "data": null
}
```

### 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
